# AI Сервис — дочерняя тема для WordPress

## Описание
Тема для сервиса генерации карточек товаров через ИИ.

## Структура
- `theme.json` — глобальные стили (фиолетовый #8B5CF6, бирюзовый)
- `templates/front-page.html` — главная страница с 3 шагами
- `parts/header.html` и `footer.html` — шапка и подвал

## Как установить
1. Скопировать папку `ai-service-child` в `/wp-content/themes/`
2. Активировать тему в админке WordPress
